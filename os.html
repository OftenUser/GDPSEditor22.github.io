FAQ / Help

1. What Is A GDPS?

It is a Geometry Dash Private Server, where we have our own levels, our own in-game moderators, working Top 100 and so on.

2. Why Is This GDPS Special?

This GDPS runs on the most recent 2.2 leak, it happened in December 2019 when RobTop leaked new contents of the 2.2 Update on GD LIte.

3. From Which Platforms Can I Play The GDPS?

The answer is related to the question above, the leak happened only on the Play Store which means this GDPS is only playable on Android and on other platforms where Android can be emulated such as PC using Bluestacks 5 Emulator.

4. What Kind Of New Features Does This GDPS Offer That Are Not Already In Others?

The biggest one is the Platformer Mode, as you can see on the Sneak Peek by RobTop himself allows you to create 2D platformer-gameplay. This GDPS has also a lot of new 2.2 Editor features like the Particle Editor, Swing Copter Game-Mode, and new triggers like Song Trigger, Timewarp Trigger, etc.
